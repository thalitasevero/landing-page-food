# 🌐 Landing Page Responsiva e Animada

Bem-vindo ao repositório da minha Landing Page!

## 📋 Descrição

A página possui um design totalmente responsivo, garantindo uma ótima experiência em diversos dispositivos. Além disso, incorpora animações sutis que tornam a navegação mais fluida e agradável.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica e organização do conteúdo.
- **CSS3**: Estilização da página com flexbox, grid layout e técnicas de design responsivo.
- **JavaScript**: Implementação de animações interativas.

## 🚀 Funcionalidades

- **Design Responsivo**: Adaptável a diferentes tamanhos de tela, proporcionando uma experiência consistente em dispositivos móveis, tablets e desktops.
- **Navbar Interativa**: Inclui links para as seções **Início**, **Cardápio** e **Avaliações**.
- **Section**: Exibe informações importantes de forma visualmente atraente no topo da página.

## 🎨 Estilo e Animações

O visual da página foi construído com:
- **Google Fonts**: Utilização de fontes customizadas para melhorar a estética do texto.
- **ScrollReveal**: Biblioteca de animações que revela conteúdo à medida que o usuário navega pela página.
- **jQuery CDN**: Biblioteca JavaScript para simplificar a manipulação do DOM e criar interações dinâmicas.
